## 🚀 快速开始

### 添加账号（快速方式）
```
1. 在浏览器中登录平台
2. 打开插件 → 账号管理 → 添加账号
3. 选择平台
4. 点击"快速添加（已登录）"
5. 完成！
```

### 添加账号（引导方式）
```
1. 打开插件 → 账号管理 → 添加账号
2. 选择平台
3. 点击"引导登录"
4. 在打开的页面完成登录
5. 插件自动检测并保存账号
6. 完成！
```

### 发布文章
```
1. 编辑文章 → 填写标题和内容
2. 点击"发布"按钮
3. 选择要发布的平台
4. 点击"发布到 X 个平台"
5. 等待发布完成
6. 查看发布结果
```

## 🔍 故障排查速查表

### 问题：账号添加失败

| 症状 | 原因 | 解决方案 |
|------|------|---------|
| 显示"未登录" | 浏览器中未登录 | 先在浏览器中登录平台 |
| 显示"超时" | 登录检测超过3分钟 | 重试或检查网络 |
| 显示"无法获取用户信息" | API 调用失败 | 检查网络或平台状态 |
| 显示"权限不足" | 插件权限问题 | 检查浏览器权限设置 |

### 问题：发布失败

| 症状 | 原因 | 解决方案 |
|------|------|---------|
| 显示"账号未登录" | 账号登录状态过期 | 刷新账号信息或重新添加 |
| 显示"标题为空" | 未填写标题 | 填写文章标题 |
| 显示"内容为空" | 未填写内容 | 填写文章内容 |
| 显示"页面加载超时" | 平台响应慢 | 重试或检查网络 |
| 显示"元素未找到" | 平台页面结构变化 | 等待适配器更新 |

## 💡 使用技巧

### 技巧 1：快速复制文章内容
```
编辑文章页面
→ 点击标题框右上角的复制按钮
→ 复制标题到剪贴板
→ 点击正文框右上角的复制按钮
→ 复制正文到剪贴板
```

### 技巧 2：批量发布到多个平台
```
1. 编辑文章
2. 点击"发布"
3. 点击"全选"选择所有平台
4. 点击"发布到 12 个平台"
5. 一次性发布到所有平台
```

### 技巧 3：刷新账号信息
```
账号管理
→ 找到要刷新的账号
→ 点击"刷新"按钮
→ 自动更新用户信息（昵称、头像等）
```

### 技巧 4：禁用不需要的账号
```
账号管理
→ 找到要禁用的账号
→ 点击右侧的开关
→ 账号将不会出现在发布列表中
```

### 技巧 5：查看发布进度
```
发布文章后
→ 自动跳转到"任务中心"
→ 查看实时发布进度
→ 查看每个平台的发布结果
```

## 🔐 安全建议

### Cookie 和登录状态
- ✅ 插件使用 Chrome 的 Cookie API 检测登录状态
- ✅ 不会存储或上传任何密码
- ✅ 只读取必要的认证 Cookie
- ✅ 所有数据都存储在本地

### 账号信息
- ✅ 账号信息存储在本地数据库
- ✅ 不会上传到任何服务器
- ✅ 可以随时删除账号
- ✅ 支持导出和备份

### 文章内容
- ✅ 文章内容存储在本地
- ✅ 发布时才会上传到平台
- ✅ 支持离线编辑
- ✅ 自动保存草稿

## 🎉 常见问题 (FAQ)

**Q: 支持多少个平台？**
A: 支持全部12个平台：掘金、CSDN、知乎、微信公众号、简书、博客园、51CTO、腾讯云、阿里云、思否、B站、开源中国。

**Q: 支持离线编辑吗？**
A: 支持。所有文章都存储在本地，可以离线编辑。

**Q: 支持导出文章吗？**
A: 目前支持复制文章内容。完整的导出功能正在开发中。

**Q: 支持定时发布吗？**
A: 目前不支持。立即发布功能已完成，定时发布功能正在开发中。

**Q: 支持图片上传吗？**
A: 支持。发布时会自动上传图片到平台。

**Q: 支持 LaTeX 公式吗？**
A: 部分支持。不同平台对 LaTeX 的支持不同，插件会自动转换。

**Q: 如何删除账号？**
A: 账号管理 → 找到账号 → 点击"删除"按钮。

**Q: 如何禁用账号？**
A: 账号管理 → 找到账号 → 点击右侧的开关。

**Q: 如何更新账号信息？**
A: 账号管理 → 找到账号 → 点击"刷新"按钮。

**Q: 发布失败怎么办？**
A: 查看任务中心的错误信息，根据错误提示排查问题。

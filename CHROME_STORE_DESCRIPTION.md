# Chrome Web Store 概述

一站式多平台内容同步助手，一次编辑，处处发布：支持掘金、CSDN、知乎、微信公众号、简书、博客园等 17+ 主流博客平台一键同步发布

🔥 SyncCaster 是内容创作者的全能发布助手。
编辑一次，一键同步到所有平台。
告别重复的复制粘贴，让创作更专注。

💪 主要功能：

👉 一键多平台发布
✔️ 17+ 平台支持：掘金、CSDN、知乎、微信公众号、简书、博客园、51CTO、腾讯云、阿里云、思否、哔哩哔哩、开源中国、今日头条、InfoQ、百家号、网易号、Medium
✔️ 智能格式转换：自动适配各平台的 Markdown/富文本编辑器
✔️ 样式完整保留：微信公众号发布时完整保留渲染样式
✔️ 标签页分组：同步的标签页自动归入分组，便于管理

👉 内置 Markdown 编辑器
✔️ 实时预览：所见即所得的编辑体验
✔️ 代码高亮：支持多种编程语言语法高亮
✔️ LaTeX 公式：完美支持数学公式渲染
✔️ Mermaid 图表：流程图、时序图等图表支持
✔️ 多种主题：丰富的样式主题可选

👉 智能内容采集
✔️ 网页采集：从任意网页智能提取文章内容
✔️ 格式识别：自动识别标题、正文、图片、公式
✔️ 一键转换：将采集内容转换为统一 Markdown 格式

👉 账号管理
✔️ 登录检测：自动检测各平台登录状态
✔️ 过期预警：Cookie 即将过期时提前提醒
✔️ 一键刷新：快速刷新所有账号状态
✔️ 快捷登录：支持一键跳转重新登录

👉 任务管理
✔️ 发布队列：批量发布任务排队执行
✔️ 进度追踪：实时查看发布进度
✔️ 详细日志：完整记录发布过程
✔️ 失败重试：发布失败支持重试

🔒 隐私保护：
✔️ 完全本地运行，不收集任何用户数据
✔️ 不存储您的账号密码
✔️ 开源透明，代码可审查

💻 使用方法：
🔸 点击"添加到 Chrome"按钮安装扩展
🔸 点击扩展图标打开 SyncCaster
🔸 在各平台登录您的账号
🔸 编辑或采集文章内容
🔸 选择目标平台，一键发布！

❓ 常见问题：

📌 支持哪些平台？
- 目前支持 17+ 主流博客平台，包括掘金、CSDN、知乎、微信公众号等，未来将支持更多平台。

📌 需要付费吗？
- 完全免费，无任何付费功能。

📌 我的数据安全吗？
- 所有数据都存储在本地浏览器中，不会上传到任何服务器。

📌 为什么需要这么多权限？
- 扩展需要访问各博客平台页面来执行自动化发布操作，这是实现一键发布的必要权限。
